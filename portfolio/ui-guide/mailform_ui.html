MAIL FORM

시안이 완료된 후, 아래와 같은 형식으로 메일을 작성하여 공유한다.

메일 작성

메일의 제목은 이해하기 쉽도록 "[공유명][프로젝트명] 작업 제목"의 규칙으로 작성한다.

- 공유명 : 디자인 공유 / 디자인 수정
- 프로젝트명 : 우리펜션 / yt corp / 관리자
- 작업 제목 : 공유하는 작업에 대한 제목

시안은 메일에 첨부하지 않으며, 공용 폴더에 업로드 후 해당 파일 경로를 공유하는 형식으로 작성한다.

완료 공유
· 제목 :  [디자인 공유][우리펜션] 회원가입 리뉴얼

· 받는 사람 :  마크업 담당자

· 참조 :  기획 담당자, 해당 업무의 리더

· 내용 :  '우리펜션 - 회원가입 리뉴얼' 작업이 완료되어 공유드립니다.
           경로는 아래와 같습니다.

           --------------------------------------------------------------------------------------------------------

           - 로그인 : 드라이브 > UX팀 > 회원가입 > 로그인.psd
           - 아이디/비밀번호 찾기 : 드라이브 > UX팀 > 회원가입 > 아이디/비밀번호 찾기.psd
           - 회원가입 폼 : 드라이브 > UX팀 > 회원가입 > 회원가입 폼.psd

           --------------------------------------------------------------------------------------------------------

           작업시 문의사항은 답변이나 메신저주시면 감사하겠습니다.
수정 요청

최신 버전의 파일명에 대한 내용을 추가한다.


· 제목 :  [디자인 수정][우리펜션] 회원가입 리뉴얼

· 받는 사람 :  마크업 담당자

· 참조 :  기획 담당자, 개발 담당자, 해당 업무의 리더

· 내용 :  '우리펜션 - 회원가입 리뉴얼' 작업이 수정되어 재공유드립니다.
