FONT (system)

본문, 서브 페이지 타이틀 등 거의 모든 텍스트 요소에 쓰이며, 웹 접근성을 높이고 다양한 기기에
대응해나갈 수 있게 끔 차후에는 비중을 늘려나갈 수 있도록 한다.

character

맑은 고딕(Malgun Gothic) - Regular
맑은 고딕(Malgun Gothic) - Bold

color

기본 텍스트 - #000000 #323232 #626262 #898989
링크 - #4675bb
강조할 부분, 가격표시 - #e20000
※ mouseover시 페이지 이동 텍스트는 하단 라인으로 구분한다.

FONT (image)

메인 페이지의 타이틀/배너/메뉴/포인트가 되는 문구에 사용하며, 웹 접근성을 높이고 다양한 기기에
대응해나갈 수 있게끔 차후에는 비중을 줄여나갈 수 있도록 한다.

character

국문 - 윤고딕 520~550




영문 - Gotham Bold / Medium / Light




포인트 - 나눔펜글씨

button

이미지-텍스트 버튼을 사용하며, 가이드 PSD의 버튼을 수정해서 사용하는 것을 원칙으로 한다.
단, 보조 버튼에 한해서 담당 디자이너가 필요하다고 판단할 때 메인 컬러를 응용해 코드로 된 버튼 요소를 새로 만들 수 있다.
